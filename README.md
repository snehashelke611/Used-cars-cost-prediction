# Used Car Cost Prediction

## Project Overview
This project analyzes used car data to understand how different factors such as manufacturing year, fuel type, transmission, seller type, and kilometers driven affect the selling price of cars. The project uses Python libraries for data cleaning, analysis, and visualization.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Summary of Data
- Data Visualization using Count Plots and Scatter Plots
- Correlation Analysis using Heatmap
- Analysis of factors affecting car selling prices

## Dataset
Used Car Dataset from CarDekho containing **4340 records** and **8 features**:
- Car Name
- Manufacturing Year
- Selling Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission Type
- Owner Information

## Key Skills
- Data Analysis
- Data Visualization
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Python Programming

## Visualizations Performed
- Fuel Type Count Plot
- Transmission Type Count Plot
- Seller Type Count Plot
- KM Driven vs Selling Price Scatter Plot
- Year vs Selling Price Scatter Plot
- Correlation Heatmap

## Conclusion
This project helped in understanding the relationship between various car features and their impact on selling prices. The analysis showed that factors such as manufacturing year, kilometers driven, and transmission type significantly influence used car prices.
